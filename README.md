# nessus_parser
.nessus parser

Parsing Nessus results into docx
<br><br>

How to use:
```
git clone https://github.com/jimbet/nessus_parser.git
pip3 install python-docx
chmod +x jim_nessus_parser.py
python jim_nessus_parser.py report.nessus output-doc.docx
```
The output file will be opened automatically
